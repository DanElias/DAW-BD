Preguntas e investigación
¿Qué es entonces una base de datos - DB?
Una base de datos es una recopilación de información organizada para proporcionar un acceso rápido a los datos. La información recopilada puede estar en cualquier número de formatos (electrónicos, impresos, gráficos, de audio, estadísticos, combinaciones). Es una colección de archivos interrelacionado que contienen la menor repetición posible de datos y abarcan las necesidades de la empresa.[1]
¿En qué casos es conveniente usar bases de datos?
Cuando se quiera obtener y buscar los datos de manera eficiente, cuando estén interrelacionados, cuando se quieran hacer reportes con esos datos. Cuando quieras tener los datos de manera que siempre puedan obtenerse, que no se pierdan. Cuando tengas una gran cantidad de datos. Cuando existen múltiples usuarios que manejan estos datos al mismo tiempo y también usar una base de datos hace de un programa escalable.[2]
¿Qué es un sistema de gestión de base de datos - DBMS?
Un sistema de gestión de base de datos (DBMS) es una colección de programas que permite crear y mantener una base de datos. El DBMS es un software de propósito general, facilitando los procesos de definición, construcción, manipulación y uso compartido. [3]
Enlista y explica las funciones/responsabilidades que tienen los DBMS.
Definición: especificar los tipos de datos, estructuras y restricciones de los datos. Crea los metadatos.
Construir: guardar los datos.
Manipular: buscar y obtener datos, actualizarlos y poder generar reportes.
Compartir: múltiples usuarios pueden acceder a la base de datos simultáneamente.
Protección: protege el sistema contra errores de hardware o software y no deja que entre software malicioso o un acceso no autorizado.
Mantener: la cantidad de datos puede ir creciendo a través de los años y permite que este evolucione dependiendo de los requerimientos de la empresa o cliente.[4]
Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
En Organización Computacional creé un espejo inteligente que registraba las emociones de las personas a través de una API de Microsoft de reconocimiento facial de emociones. Con una base de datos y un dbms hubiera podido guardar los datos de las emociones que el paciente tenía cada día para generar reportes para que un psicólogo tuviera información extra para realizar un diagnóstico y monitorear el progreso del paciente.
