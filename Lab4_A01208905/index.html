A Trivial Problem

Mr. Santa asks all the great programmers of the world to solve a trivial problem. He gives them an integer m and asks for the number of positive integers n, such that the factorial of n ends with exactly m zeroes. Are you among those great programmers who can solve this problem?

Input
The only line of input contains an integer m (1 ≤ m ≤ 100 000) — the required number of trailing zeroes in factorial.

Output
Print the values of n such that the factorial of n ends with m zeroes.

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

-JavaScript no crea applets aplicaciones independientes. Puede proporcionar niveles de interactividad a páginas web.

-Java es un lenguaje de programación Orientado a Objetos, mientras que JavaScript es un lenguaje de scripting (scripting language) Orientado a Objetos.

-Con Java se pueden crear aplicaciones que se ejecutan en una máquina virtual o navegador, mientras que JavaScript sólo se ejecuta en un navegador.

-El código Java debe compilarse, mientras que JavaScript está todo en texto.

-JavaScript es un lenguaje interpretado mientras que Java es un lenguaje compilado.

-Requieren diferentes plug-ins. [1]

¿Cómo se declara una variable con alcance local dentro de una función?

Simplemente se declara pero dentro de la función, Se puede usar var para declararla. Una función que se declara dentro de una función sólo funcionará dentro de esa función y funciones que estén dentro de la misma función. En JavaScript, si se declara una función adentro de un bloque como for, toda la función tendrá acceso a ella.[4]
¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables que se declaran fuera de una función son globales y son accesibles desde cualquier parte del código. Si se tienen muchas variables globales es posible confundirse y volver a declarar la misma variable en otra parte del código, además de que hay menos control sobre qué cosas están afectando a la variables. Además JS tarda más en encontrar a las variables globales.[5]

Se puede usar match() y se puede utilizar para buscar palabras o expresiones en una página web de una forma rápida. Como por ejemplo si necesito navegar en una página web y buscar algo en la sección de búsqueda.[6]
